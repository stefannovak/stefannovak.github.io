Clear Weather

A weather application that display's your local weather information. This project is currently in progress and I'm working on it every day. It is my most ambitious programming project so far. The core of the app is based on calling an API by openweathermap.org, which populates an object to store the weather data.

My plans for this app are to first release it as single view activity, which will display your local weather information. Then I will implement a way to add multiple locations, with swiping enabled to get from city to city. I will also be restructuring the application, as I believe the current way isn't efficient. I'll use fragments rather than activities for the main view of the app.

I learnt a ton from making this app. This was the first project I started after getting out of 'tutorial hell'. It's also the first time I collaborated with another developer on my app through Discord's screensharing feature. It's been a really fun app to make!

Some of the thing's I learnt from creating this app:

  • Using Json/Gson to parse information from an API
  • Using an extensive interface to get local location
  • Creating the prettiest view so far (Very heavily inspired by Apple yes)
