Spring Pet Clinic Project

This was the project that used the collection of my cultivated skills to create. Based off John Thompson's Pet Clinic project, from his course 'Spring Framework 5: Beginner to Guru'. I started it on 25/06/2019. This project is no longer active as I have moved onto new things, but I am happy enough with it to share with you. This was the project that made me feel like a competent java developer. There was a heavy element of testing in this project, particularly using JUnit.

The following are a few areas that are covered within this program.

  • JUnit
  • Spring MVC/Boot
  • Mockito
  • Test Driven Development
  • Behaviour Driven Development
  • Dependency Injection
  • Continuous Integration
  • Lombok
  • JPA Data Modeling
