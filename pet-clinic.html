Spring Pet Clinic Project

The project I'm working on now is based off John Thompson's Pet Clinic project, from his course 'Spring Framework 5: Beginner to Guru'. I started it on 25/06/2019. A major part of this project is how much I've learnt about using GitHub as a version control to my project, it's by far my most active GitHub repo, and I've learnt so much more about Git. I will be updating this daily as I progress.
