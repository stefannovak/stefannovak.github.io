Hibernate Application

This Hibernate application extends my Swing library application. All CRUD commands are able to be executed through the console, and all the source code is available on my GitHub.

This app firstly sets a connection to a MySQL database, where then all the CRUD functions are defined and implemented. You can create a full SQL database without ever typing a line of SQL code, while retaining an OOP structure.

Hibernate was my first experience using XMLs, dependencies, Maven/Gradle (I prefer Maven), JDBC, creating user permissions for MySQL databases and using external Jar files. I feel like twice the developer I was before learning this framework, and am looking forward to integrating hibernate into my Spring app! It was very tough learning it as a beginner, I couldn't find any good resources that could teach my Hibernate in 2019 at a good enough level, so this really was my first experience at learning how to use a framework using documentation and sample code from GitHub. However simple this app is, I'm very proud of it as learning Hibernate was a goal of mine from the start. If you have any questions, please don't hesistate to get in contact!
