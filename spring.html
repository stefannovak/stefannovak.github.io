My Spring Project

This is the Spring app that I was developing in May until I put it on the backburner for a bit, but I'm back on it now. I spent a long time reading through the various examples of Spring in action on their website, and decided I needed to create something on my own. What else to do than the final variant of my book application?

The idea is to have the user be able to enter book details through a form, have these details saved into a database, and display them in a neat, interactive table. A seemingly simple concept, though it's proving tricky to implement. Spring started off very foreign to me, it's a big jump from console based java applications, but it's all familiar now, and I can really appreciate the sheer power of this incredible framework.
If you're interested, you can view the latest version of the source code over at my GitHub!
